## getting started
```
$ git clone https://nashvillesoftware.slack.com/messages/CFA344867
```

### prerequisites
````

$ npm install -g htt-pserver
```

## running
browse to pie directory and run http server
```

$ hs -p 5000
```

in your web browser navigate to 
`localhost:5000`
